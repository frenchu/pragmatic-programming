Samba server setup on Ubuntu

January 21, 2024 One-minute read
tools
samba • services • networking • admin

Install samba Link to heading

sudo apt update
sudo apt install samba

Add samba user for you Link to heading

sudo smbpasswd -a $USER
sudo usermod -a -G sambashare $USER
net usershare add priv ~/Documents/priv "My Documents" $USER:f,everyone:d guest_ok=n

Edit config Link to heading

cat /etc/samba/smb.conf
sudo service smbd restart

Add accounts for others Link to heading

sudo useradd -M -G sambashare karen
sudo passwd -l karen
net usershare add documents ~/Documents/shared "Shared Documents" $USER:f,karen:f,everyone:d guest_ok=n

Verify users and shares Link to heading

sudo pdbedit -L -v
net usershare info --long

Configure firewall Link to heading

sudo ufw allow from 192.168.10.0/24 to any app samba
sudo ufw reload
